# Feature Implementation Workflow

1.  **Analyze**: Understand the feature from project requirements and UI designs.
2.  **Design**: Plan the implementation according to [MVVM Architecture](../../shared/skills/mvvm-architecture-implementation.md).
3.  **Implement**: Write the code using Jetpack Compose and Kotlin, adhering to [Jetpack Compose Best Practices](../../shared/skills/jetpack-compose-best-practices.md).
4.  **Test**: Collaborate with the [Tester Agent](../tester-agent/agent.md) to verify with unit and UI tests.
5.  **Self-Review**: Use the [Reviewer Persona](../code-reviewer-agent/agent.md) to check your own work before completion.
