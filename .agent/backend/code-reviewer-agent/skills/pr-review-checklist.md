# PR Review Checklist

1.  **Requirement Check**: Does the code fulfill the requirements in [requirements.md](../../../requirements.md)?
2.  **NestJS Standards**: Does it follow the [shared NestJS skills](../../shared/skills/nestjs-best-practices.md)?
3.  **Security**: Are there any obvious security flaws?
4.  **Testing**: Are there corresponding tests for the new logic?
5.  **Documentation**: Is the code well-documented where necessary?
