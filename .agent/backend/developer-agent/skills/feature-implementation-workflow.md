# Feature Implementation Workflow

1.  **Analyze**: Understand the feature from [requirements.md](../../../requirements.md).
2.  **Design**: Plan the implementation according to [Architecture Standards](../../shared/skills/arch-best-practices.md).
3.  **Implement**: Write the code following [NestJS Best Practices](../../shared/skills/nestjs-best-practices.md).
4.  **Test**: Verify with unit and E2E tests.
5.  **Self-Review**: Use the [Reviewer Persona](../agent.md) to check your own work.
