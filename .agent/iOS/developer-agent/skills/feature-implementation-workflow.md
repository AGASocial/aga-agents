# Feature Implementation Workflow

1.  **Analyze**: Understand the feature from requirements and acceptance criteria outlined in the technical task. Identify any missing information before starting.
2.  **Design**: Plan the implementation according to [Architecture Best Practices](../../shared/skills/ios-arch-best-practices.md). Determine the necessary components, state management, and data flow.
3.  **Implement**: Write the code following [Swift Best Practices](../../shared/skills/swift-best-practices.md) and [UI Best Practices](../../shared/skills/ios-ui-best-practices.md). Keep methods small and focused.
4.  **Test**: Verify the implementation with unit tests covering edge cases and happy paths.
5.  **Self-Review**: Use the Reviewer Persona to check your own work against the PR Review Checklist. Refactor if necessary.
