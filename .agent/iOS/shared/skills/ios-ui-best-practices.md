# iOS UI Best Practices

## General Guidelines
1. **Human Interface Guidelines (HIG)**: Adhere to Apple's HIG for consistent and familiar user experiences.
2. **Responsive Design**: Use Auto Layout or SwiftUI to ensure your app looks great on all device sizes and orientations (iPhone, iPad).
3. **Accessibility (a11y)**: Support VoiceOver, Dynamic Type, and high contrast modes. Ensure tap targets are at least 44x44 points.
4. **Animations**: Use animations purposefully to guide the user, but keep them subtle and ensure they don't block interaction.
5. **Dark Mode**: Always design and test for both Light and Dark mode appearances.
