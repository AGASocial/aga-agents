# Android Design Guidelines

When creating technical tasks for Android, ensure you consider the following:
- **Architecture:** Follow established patterns like MVVM (Model-View-ViewModel).
- **UI Layouts:** Specify new Activities, Fragments, and XML layouts or Jetpack Compose UI elements.
- **Data Layer:** Detail changes to the repository pattern implementations, Room database (if applicable), and remote data sources.
- **Network Calls:** Define the Retrofit interfaces or API calls required.
- **Permissions:** Note any new Android permissions that need to be requested from the user.
