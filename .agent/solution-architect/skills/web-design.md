# Web (Frontend) Design Guidelines

When creating technical tasks for the Web frontend, ensure you consider the following:
- **UI Components:** Identify new or modified React/Vue/Angular components (or equivalent framework).
- **API Integrations:** Specify the backend APIs that need to be called and how data will be consumed.
- **State Management:** Detail how the application state will be updated (e.g., Redux, Context API, component state).
- **Routing:** Define any new routes or navigation paths.
- **Styling & UX:** Note any specific styling requirements, animations, or user experience considerations.
